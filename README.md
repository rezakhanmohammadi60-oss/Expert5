# ExpertXAU_M1

این مخزن شامل یک اکسپرت اولیه MQL5 برای XAUUSD در تایم‌فریم M1 است. نسخه‌ی 1.1 لاگ‌های تصمیم‌گیری (debug prints) را اضافه کرده تا در هنگام بک‌تست و اجرا بتوانید دقیقاً بدانید که چرا اکسپرت تصمیم به باز یا عدم باز کردن پوزیشن گرفته است.

قابلیت‌های نسخه 1.1

- لاگ‌های DetectMarketCycle: وقتی بازار رنج، کانال یا اسپایک تشخیص داده می‌شود، اطلاعات ATR/ADX/کندل چاپ می‌شود.
- لاگ‌های EntryCheck: SL/TP محاسبه‌شده، مقدار لات محاسبه‌شده، ریسک بالقوه و درصد ریسک پس از باز شدن چاپ می‌شود.
- لاگ ارسال سفارش (SendTradeRequest): پارامترهای درخواست سفارش و نتیجه ارسال چاپ می‌شود.
- لاگ شروع روز/هفته/ماه برای رفرنس بالانس.

راهنمای تست دیباگ

- پارامترهای پیشنهادی برای تست را طبق README نسخه اولیه اعمال کن.
- پس از اجرای Strategy Tester، پنجره Experts را باز کن و لاگ‌ها را برای خطاها یا پیام‌های تصمیم‌گیری بررسی کن.
- اگر پیام‌هایی می‌بینی که نشان می‌دهد denom در محاسبه لات صفر است، آن‌وقت لازم است مقادیر tickSize و tickValue نماد XAUUSD در بروکر خود را بررسی کنیم.
