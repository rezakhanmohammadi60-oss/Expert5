# ExpertXAU_M1

این مخزن شامل یک اکسپرت اولیه MQL5 برای XAUUSD در تایم‌فریم M1 است. اکسپرت ماژولار نوشته شده و شامل موارد زیر می‌باشد:

- تشخیص سیکل بازار: رنج / کانال / اسپایک (بر اساس ATR, ADX, اندازه کندل)
- موتور ورود: ورود در میکروکانال یا اولین پول‌بک پس از اسپایک (ساده‌سازی شده)
- مدیریت ریسک: ریسک هر معامله 1%، حداکثر ریسک باز همزمان 2%، حداکثر 3 معامله باز همزمان
- مدیریت دراودان: توقف معامله در صورت رخ دادن دراودان روزانه/هفتگی/ماهانه (بر اساس Balance)
- فیلتر زمانی: جلوگیری از معامله 15 دقیقه قبل/بعد باز شدن نیویورک (قابل تنظیم)
- تریلینگ شرطی: فعال بعد از فاصله مشخص و حرکت قدرتمند (ATR-based)
- پنل اطلاعات ساده (Comment روی چارت)
- همه کامنت‌ها به فارسی هستند

نکات مهم قبل از استفاده

1. این نسخه اولیه است و برخی قابلیت‌ها (مانند چک اخبار ForexFactory و استفاده از DOM) به صورت انتخابی پیاده‌سازی شده‌اند. فیچر اخبار با استفاده از WebRequest در صورت فعال‌سازی می‌تواند کار کند؛ اما در حال حاضر InpUseNewsFilter به صورت پیش‌فرض false است تا اگر نمی‌خواهید WebRequest را فعال کنید مشکلی پیش نیاید.

2. قبل از اجرای روی حساب واقعی، حتماً در Strategy Tester با دیتای مناسب XAUUSD و تایم‌فریم M1 تست کامل انجام دهید.

3. پارامترهای مهم را می‌توانید در پنجره اکسپرت تغییر دهید (Inputs): ATR Period, Spike multiplier, ریسک، Slippage، تریلینگ و غیره.

4. توجه: مقدار تبدیل لات/ریسک برای XAUUSD بسته به کارگزاری متفاوت است (نمونه tick value و tick size). لطفاً پس از بارگذاری اکسپرت، با یک حساب دمو و تنظیمات بروکر خود مقدار lot حداقلی و محاسبات ریسک را بررسی کنید.

فایل‌ها

- ExpertXAU_M1.mq5: سورس اکسپرت

نحوه استفاده

1. شاخه‌ای که ساخته شده: feature/expert-mql5
2. فایل ExpertXAU_M1.mq5 را در پوشه MQL5/Experts کپی کنید و با MetaEditor باز کنید.
3. برای استفاده از فیچر اخبار (ForexFactory) باید دامنه forexfactory.com را در Tools->Options->Expert Advisors->Allow WebRequest for listed URL وارد کنید.
4. اکسپرت را Compile کنید و در Strategy Tester یا چارت XAUUSD M1 اجرا کنید.

سلب مسئولیت

این اکسپرت یک نسخه اولیه/نمونه است و تضمین سود نمی‌دهد. استفاده در حساب واقعی بدون تست کافی ریسک بالایی دارد.
